Getting Started

Connect to Kairos with ease and begin your first adventure.

Download the Client

Get the latest Kairos client from the official download page and verify the files before running the launcher.

Pro Tip

Use the recommended client version to avoid compatibility issues with custom server scripts.

Create Your Account

Register a new account using the server's account creation system, then secure it with a strong password.

Connecting to the Server

Update your client settings with the Kairos server address and enter the game to begin your journey.

Need help?

Visit the FAQ or Commands page for quick troubleshooting commands.
